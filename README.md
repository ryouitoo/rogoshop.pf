# <ロゴショップ>

##サイト概要
###サイトテーマ
<...>ロゴショップ


###テーマを選んだ理由
<...>友人が趣味でロゴを製作しており、ロゴを販売できるサイトがあれば
面白いし、制作した物をまとめらられるので便利だと考えた。>

###ターゲットユーザー
<...>若者やイラストデザイナー

###主な利用シーン
<...>自分オリジナルのロゴが欲しい。またそれをステッカーや服にプリントする。
販売者側は作品を販売でき、作成したデザインを管理できる。

<...>
<1.>言語:HTML.CSS.JavaScript.Ruby
<2.>フレームワーク：Ruby on Rails
