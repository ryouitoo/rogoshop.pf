# <ロゴショップ>
<dl>
 ###<サイト概要>
 <dt>サイトテーマ</dt>
  <dd>ロゴショップ<dd>
 
 <dt>テーマを選んだ理由</dt>
  <dd>友人が趣味でロゴを製作しており、ロゴを販売できるサイトがあれば
      面白いし、制作した物をまとめらられるので便利だと考えた。</dd>
 
 <dt>ターゲットユーザー</dt>    
  <dd>若者やイラストデザイナー</dd>
 
 <dt>主な利用シーン</dt>
  <dd>自分オリジナルのロゴが欲しい。またそれをステッカーや服にプリントする。
      販売者側は作品を販売でき、作成したデザインを管理できる。</dd>
      
 <dt>開発環境</dt>
  <dd>言語：  HTML.CSS.JavaScript.Ruby</dd>
  <dd>フレームワーク:  Ruby on Rails</dd>
  <dd>IDE:  Cloud9</dd>

</dl>
